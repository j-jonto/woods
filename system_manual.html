🏭 نظام إدارة المخزون والإنتاج

دليل شامل لتشغيل وإدارة النظام

📋 نظرة عامة على النظام

هذا النظام مصمم لإدارة شاملة للمخزون والإنتاج والمبيعات والمشتريات. يتضمن جميع الوظائف الأساسية المطلوبة لإدارة مصنع أو شركة تجارية.

📦 إدارة المخزون

تتبع المخزون، معاملات الاستلام والصرف، تنبيهات المخزون المنخفض

🏭 إدارة الإنتاج

أوامر الإنتاج، قوائم المواد، تتبع التكاليف، مراكز العمل

💰 إدارة المبيعات

أوامر البيع، العملاء، المندوبين، حساب الأرباح والخسائر

🛒 إدارة المشتريات

أوامر الشراء، الموردين، فواتير الشراء، تتبع المدفوعات

🚀 بدء تشغيل النظام

الخطوات الأولية

  1. إنشاء المستخدمين: إضافة مستخدمين للنظام مع صلاحيات مناسبة
  2. إعداد البيانات الأساسية: إنشاء المستودعات، مراكز العمل، فئات المنتجات
  3. إضافة المنتجات: إنشاء المواد الخام والمنتجات النهائية
  4. إعداد قوائم المواد: تحديد المواد المطلوبة لكل منتج نهائي
  5. إضافة العملاء والموردين: إنشاء قاعدة بيانات العملاء والموردين
⚠️ مهم: يجب إكمال الخطوات بالترتيب المذكور لضمان عمل النظام بشكل صحيح.

👥 إدارة المستخدمين

إنشاء مستخدم جديد

زر "إضافة مستخدم جديد":
  • أدخل اسم المستخدم والبريد الإلكتروني
  • حدد كلمة مرور قوية
  • اختر الصلاحيات المناسبة (مدير، محاسب، مستخدم)
  • اضغط "حفظ"

إدارة الصلاحيات

  • مدير: صلاحيات كاملة على جميع الوحدات
  • محاسب: إدارة المدفوعات والحسابات والتقارير
  • مستخدم: عرض البيانات وإدخال المعاملات الأساسية

📦 إدارة المخزون

إنشاء مستودع جديد

زر "إضافة مستودع":
  • أدخل رمز المستودع (مثل: WH001)
  • أدخل اسم المستودع
  • أدخل موقع المستودع
  • حدد حالة النشاط
  • اضغط "حفظ"

إضافة منتج جديد

زر "إضافة منتج":
  • أدخل رمز المنتج والباركود
  • أدخل اسم المنتج
  • اختر نوع المنتج (مواد خام، منتج نهائي، سلعة)
  • اختر فئة المنتج
  • أدخل وحدة القياس
  • أدخل التكلفة المعيارية وسعر البيع
  • حدد نقطة إعادة الطلب والكمية
  • اضغط "حفظ"

معاملات المخزون

سيناريو استلام بضاعة

  1. اذهب إلى "معاملات المخزون"
  2. اضغط "إضافة معاملة"
  3. اختر نوع المعاملة: "استلام"
  4. حدد المنتج والمستودع
  5. أدخل الكمية وتكلفة الوحدة
  6. أدخل رقم المرجع والوصف
  7. اضغط "حفظ"

سيناريو صرف بضاعة

  1. اذهب إلى "معاملات المخزون"
  2. اضغط "إضافة معاملة"
  3. اختر نوع المعاملة: "صرف"
  4. حدد المنتج والمستودع
  5. أدخل الكمية
  6. أدخل رقم المرجع والوصف
  7. اضغط "حفظ"

🏭 إدارة الإنتاج

إنشاء قائمة مواد (BOM)

زر "إضافة قائمة مواد":
  • اختر المنتج النهائي
  • اختر المواد الخام المطلوبة
  • أدخل الكمية المطلوبة من كل مادة
  • اضغط "حفظ"

إنشاء أمر إنتاج

زر "إضافة أمر إنتاج":
  • أدخل رقم أمر الإنتاج
  • اختر المنتج المراد إنتاجه
  • اختر قائمة المواد (اختياري)
  • اختر مركز العمل (اختياري)
  • أدخل الكمية المطلوبة
  • حدد تاريخ البدء والانتهاء
  • أدخل ملاحظات (اختياري)
  • اضغط "حفظ"

مراحل الإنتاج

سيناريو بدء الإنتاج

  1. اذهب إلى أمر الإنتاج
  2. اضغط "بدء الإنتاج"
  3. سيتم خصم المواد الخام تلقائياً من المخزون
  4. تتغير حالة الأمر إلى "قيد التنفيذ"

سيناريو إكمال الإنتاج

  1. اذهب إلى أمر الإنتاج
  2. اضغط "إكمال الإنتاج"
  3. أدخل الكمية الفعلية المنتجة
  4. اختر المستودع للاستلام
  5. اضغط "حفظ"
  6. سيتم إضافة المنتج النهائي إلى المخزون تلقائياً

💰 إدارة المبيعات

إضافة عميل جديد

زر "إضافة عميل":
  • أدخل رمز العميل
  • أدخل اسم العميل وبيانات الاتصال
  • أدخل العنوان
  • حدد حد الائتمان
  • حدد حالة النشاط
  • اضغط "حفظ"

إنشاء أمر بيع

زر "إضافة أمر بيع":
  • أدخل رقم أمر البيع
  • اختر العميل
  • حدد تاريخ الطلب والتسليم
  • اختر نوع الدفع (نقدي/آجل)
  • اختر مندوب المبيعات (اختياري)
  • أضف المنتجات المطلوبة مع الكميات والأسعار
  • أدخل ملاحظات (اختياري)
  • اضغط "حفظ"

إدارة مندوبي المبيعات

زر "إضافة مندوب":
  • أدخل بيانات المندوب
  • حدد نسبة العمولة
  • حدد حالة النشاط
  • اضغط "حفظ"

إضافة إيصال قبض

سيناريو دفع العميل

  1. اذهب إلى صفحة العميل
  2. اضغط "إضافة إيصال قبض"
  3. أدخل مبلغ الدفع
  4. حدد تاريخ الدفع
  5. أدخل رقم المرجع والملاحظات
  6. اضغط "حفظ"
  7. سيتم عرض الإيصال مع إمكانية الطباعة

📊 التقارير والتحليلات

تقارير المخزون

  • تقرير المخزون الحالي: يعرض المخزون المتوفر لكل منتج
  • تقرير حركة المخزون: يعرض جميع معاملات المخزون
  • تقرير المنتجات منخفضة المخزون: تنبيهات للمنتجات التي تحتاج طلب

تقارير المبيعات

  • تقرير المبيعات: إجمالي المبيعات والأرباح
  • تقرير العملاء: تحليل أداء العملاء
  • تقرير مندوبي المبيعات: أداء المندوبين والعمولات

تقارير الإنتاج

  • تقرير أوامر الإنتاج: حالة أوامر الإنتاج
  • تقرير التكاليف: تحليل تكاليف الإنتاج
  • تقرير مراكز العمل: أداء مراكز العمل

🔍 السيناريوهات المتقدمة

سيناريو إنتاج كامل

  1. إنشاء قائمة مواد للمنتج النهائي
  2. إضافة المواد الخام إلى المخزون
  3. إنشاء أمر إنتاج
  4. بدء الإنتاج (خصم المواد الخام)
  5. إكمال الإنتاج (إضافة المنتج النهائي)
  6. إنشاء أمر بيع للمنتج النهائي
  7. تأكيد البيع (خصم من المخزون)

سيناريو البيع بالآجل

  1. إنشاء أمر بيع بنوع دفع "آجل"
  2. تأكيد البيع
  3. إضافة إيصال قبض عند الدفع
  4. مراجعة كشف حساب العميل

سيناريو الشراء بالآجل

  1. إنشاء أمر شراء بنوع دفع "آجل"
  2. استلام البضاعة (إضافة للمخزون)
  3. إضافة إيصال صرف عند الدفع
  4. مراجعة كشف حساب المورد

سيناريو مندوب المبيعات

  1. إنشاء مندوب مبيعات
  2. إنشاء أمر بيع مع تحديد المندوب
  3. تأكيد البيع (يتم إضافة بضاعة للمندوب)
  4. إضافة حركة دفع من المندوب
  5. مراجعة كشف حساب المندوب

⚠️ النقاط المهمة والتحذيرات

تحذيرات عامة

  • تأكد من إدخال البيانات بدقة قبل الحفظ
  • احتفظ بنسخ احتياطية من قاعدة البيانات بانتظام
  • راجع الأرصدة قبل إجراء أي معاملات
  • تأكد من توفر المخزون قبل إنشاء أوامر البيع

نصائح للاستخدام الأمثل

  • استخدم رموز واضحة ومنظمة للمنتجات والعملاء
  • حدد نقاط إعادة الطلب المناسبة للمنتجات
  • راجع التقارير بانتظام لاتخاذ القرارات
  • احتفظ بسجلات دقيقة لجميع المعاملات

مزايا النظام

  • تتبع دقيق للمخزون في الوقت الفعلي
  • حساب تلقائي للأرباح والخسائر
  • إدارة شاملة للعملاء والموردين
  • تقارير تفصيلية ومفصلة
  • واجهة سهلة الاستخدام

📞 الدعم والمساعدة

في حالة مواجهة أي مشاكل أو تحتاج مساعدة إضافية:

  • راجع هذا الدليل بعناية
  • تأكد من إدخال البيانات بالشكل الصحيح
  • تحقق من صلاحيات المستخدم
  • راجع سجلات النظام للأخطاء
